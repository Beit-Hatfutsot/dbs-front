lang

A module to handle language (currently English & Hebrew).

Installation

First include undefined javascript file in your HTML:

You can download this file from the following places:

  • Bower
    e.g. 
    bower install angular

Then load the module in your application by adding it as a dependent module:

angular.module('app', ['lang']);

With that you're ready to get started!

Module Components

Directive

Name Description
en

Shows transcluded content iff the language is set to English.
he

Shows transcluded content iff the language is set to Hebrew.

Service

Name Description
langManager

This service stores information about the current (selected) language.
