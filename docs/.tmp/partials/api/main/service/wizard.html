wizard

  1. - service in module main

The Wizard service is in charge of wizard-searching & of keeping search state.

Methods

  • clear();

    Clear wizard result and search status.

  • search();

    Triggers a wizard search using the wizard query. Sets search_status, last_search, result. Filters (live) individual data. It also handles notifications.

    Returns

    Promise

Properties

  • in_progress

    Indicates that the wizard service is busy.

  • failed

    Indicates that a wizard operation has failed.

  • query

    An object representing the current wizard query. Once a wizard search is triggered, this query will be used. The query holds two strings: a name and a place.

  • result

    The result of the last wizard search.

  • search_status

    Wizard search status.

  • last_search

    Last wizard search.
