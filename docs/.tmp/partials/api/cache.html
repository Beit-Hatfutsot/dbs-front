cache

Module to handle all cache related stuff. We might want to replace it with some of Angular's built-in cache capabilities.

Installation

First include undefined javascript file in your HTML:

You can download this file from the following places:

  • Bower
    e.g. 
    bower install angular

Then load the module in your application by adding it as a dependent module:

angular.module('app', ['cache']);

With that you're ready to get started!

Module Components

Object

Name Description
cached_items

A private object of the cache service. Serves as a fallback cache mechanism when the window.sessionStorage is not defined. It implements some methods from the sessionStorage API.

Service

Name Description
cache

Custom cache service. The cache service uses browser's sessionStorage if available, and falls back to storing cached object (cached_items) in memory, if not.
