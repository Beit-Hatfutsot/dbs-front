auth.controller:AuthCtrl

  1. - object in module auth

Controller for auth modal template. Handles auth modal scope.

Methods

  • signin();

    Calls method signin of auth service, using signin form data. Defines the success and failure messages, displayed upon modal dismissal.

  • register();

    Calls method register of auth service, using registration form data. Defines the success and failure messages, displayed upon modal dismissal.

  • dismiss();

    Dismisses the auth modal.

  • goto_register();

    Switches the auth modal to registration mode.

  • goto_register();

    Switches the auth modal to signin mode.

  • select_field(key);

    Sets the selected_field property.

    Parameters

    Param Type Details
    key String

    key of the selected field

Properties

  • lang

    Passes the lang property from langManager. Probably should be removed & replaced in the template, since there is a lang property on $rootScope, unless lang is removed from $rootScope.

  • selected_field

    Key of the currently selected field. Used to toggle input label class.

  • auth_in_progress

    Indicates that an auth request is in progress.

  • signin_data

    Object that binds to the signin form inputs.

  • register_data

    Object that binds to the registration form inputs.

  • message

    English & Hebrew messages to display to the user.

  • placeholders

    English & Hebrew form placeholders.

  • submit_values

    Engilsh & Hebrew submit button values.
